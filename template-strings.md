Single line template string
```javascript
`This is a template string`
```

Multiline template string
```javascript
let text = `This is a multiline
string that
just works!`
```

Interpolation
```javascript
let language = "JavaScript";
`I am learning ${language}`; //"I am learning JavaScript";
```

Sample usage
``` javascript
return `<tr>
        <td>${label}</td>
        <td>${value}</td>
    </tr>`
```
