## Basic syntax

```javascript
if (condition) {
    //do something
}
```

else
```javascript
if (condition) {
    //do
} else {
    //do
}
```

else if
``` javascript
if (condition) {
	//do
} else if (other condition) {
	//do
} else {
	//do
}
```
## question mark

```js
let result = condition ? value1 : value2;
```
